# EZ-AudioMate

EZ-AudioMate is a GUI-based tool for managing audio files. It allows you to:

- Resample audio files to a user-specified sampling rate
- Compress WAV files to FLAC format
- Convert FLAC files back to WAV

This Python-based application features an intuitive GUI for easy selection of input and output folders, as well as additional processing options.

## Requirements

- Python 3.11 or later
- Required Python packages: `tkinter`, `PIL`, `soundfile`, `librosa`, `soxr`

## Usage

1. Open the project folder.
2. Launch the application by running `main.py` in your Python environment or IDE.
3. Use the tabs to navigate between **Resampling**, **FLAC Conversion**, **WAV Conversion**, and **Audio Info**.
4. Select input and output folders using the “Browse” buttons.
5. Adjust settings as needed and click the action buttons to process your audio files.
